SOP Number

Objective

Qualitative and quantitative analysis of substances by measuring absorbance in the UV and visible wavelength range.

Caution

1. Use clean scratch-free cuvettes and handle carefully. 2. Do not touch optical surfaces with bare hands. 3. Warm up the instrument before use for stable readings.

Responsibility

UG & PG Students

Operating Procedure

Step I: Switch ON power. Initialization takes approx. 3 min 30 sec.

Step II: Mode selection screen appears. Available modes: 1. Photometric, 2. Spectrum, 3. Quantitation, 4. Kinetic, 5. Time Scan, 6. Multicomponent, 7. Optional Program Pack, 8. Utilities.

Step III: Press Key 1–8 to select the required mode.

Step IV: The configuration screen for the selected mode is displayed.

Photometric Mode: Set wavelength (e.g. 550.0 nm). Press F1 to select % T or Abs. Press Go to WL Key to set wavelength. Press F3 to display measurement.

Spectrum Mode: Set Meas. Mode to Abs. Scanning range 190nm–1100nm. Rec. Range 3.999–3.999. Scan Speed: Fast. Number of Scans: 1. Display Mode: Sequential. Set baseline correction before scanning.

Quantitation Mode: Set up to 3 wavelengths (max 700nm). Method: C = K.Abs + B (K=1.000, B=0.0000). Unit: mg/ml. Number of measurements: 1. Enable baseline correction. Press Start Key to measure.
