Standard Operating Procedure

Step I — Power On
Switch ON power. Instrument displays initialization program. Wait approx. 3 min 30 sec for full initialization.
Step II — Mode Selection
Mode selection screen appears with 8 modes: Photometric, Spectrum, Quantitation, Kinetic, Time Scan, Multicomponent, Optional Program Pack, Utilities.
Step III — Select Mode
Press Key 1–8 to select the required mode. The configuration screen for that mode will be displayed.
Photometric Mode
Set wavelength (e.g. 550.0 nm). Press F1 for % T or Abs. Press Go to WL Key to set wavelength. Press F3 to display measurement.
Spectrum Mode
Meas. Mode: Abs. Scanning range: 190–1100nm. Scan Speed: Fast. Scans: 1. Display: Sequential. Set baseline correction before scanning.
Quantitation Mode
Set up to 3 wavelengths (max 700nm). Method: C = K.Abs + B. Unit: mg/ml. Enable baseline correction. Press Start Key to measure.

Ask Lab Companion

INSTRUMENT: UV-Visible Spectrophotometer, Shimadzu UV-1800 SOP NO: PChem/27.01. Effective: 01/01/2026. Review: 31/12/2028. OBJECTIVE: Qualitative and quantitative analysis of substances by measuring absorbance in the UV and visible wavelength range. CAUTION: 1. Use clean scratch-free cuvettes and handle carefully. 2. Do not touch optical surfaces with bare hands. 3. Warm up the instrument before use for stable readings. RESPONSIBILITY: UG and PG Students. OPERATING PROCEDURE: Step I: Switch ON power. Initialization takes approx. 3 min 30 sec. Step II: Mode selection screen appears. Available modes: 1. Photometric, 2. Spectrum, 3. Quantitation, 4. Kinetic, 5. Time Scan, 6. Multicomponent, 7. Optional Program Pack, 8. Utilities. Step III: Press Key 1-8 to select the required mode. Step IV: The configuration screen for the selected mode is displayed. Photometric Mode: Set wavelength (e.g. 550.0 nm). Press F1 to select % T or Abs. Press Go to WL Key to set wavelength. Press F3 to display measurement. Spectrum Mode: Set Meas. Mode to Abs. Scanning range 190nm-1100nm. Scan Speed: Fast. Number of Scans: 1. Display Mode: Sequential. Set baseline correction before scanning. Quantitation Mode: Set up to 3 wavelengths (max 700nm). Method: C = K.Abs + B. Unit: mg/ml. Enable baseline correction. Press Start Key to measure. TROUBLESHOOTING — WRONG OR UNSTABLE READINGS (check in this order, most to least common): 1. DILUTION/CONCENTRATION PROBLEM (most common): If absorbance is above 2 or going up to 4, the concentration is too high and outside Beer-Lambert law linear range. Ideal absorbance is 0.2 to 0.8. Must be below 1 for calibration and linearity studies. Solution: dilute the sample further and re-measure. This is the most frequent mistake students make. 2. CUVETTE ORIENTATION WRONG (very common): Cuvette has two sides — smooth (transparent, faces the light path) and rough (frosted, where you hold it). If inserted the wrong way the light cannot pass correctly and readings will be wrong. Always insert with smooth side facing the light beam. 3. WRONG CUVETTE TYPE: Lab has two types — plastic and quartz. Never use plastic cuvettes for UV measurements below 320nm. Plastic absorbs UV light and gives completely wrong readings and bad spectra. Always use quartz cuvettes for UV range measurements. 4. BASELINE NOT SET: Always zero the instrument with a blank (pure solvent with no drug) before measuring samples. If baseline is not set, all readings will be offset and incorrect. 5. FILTRATION PROBLEM: If performing an assay, ensure the drug solution is properly filtered before measurement. Unfiltered particles scatter light and cause incorrect or unstable absorbance values. 6. WARM-UP SKIPPED: Instrument requires 3 minutes 30 seconds warm-up after switching on for stable readings. ASSAY PROCEDURE — UV SPECTROPHOTOMETRIC METHOD (GENERAL, FOR ANY API IN TABLET FORM): PRINCIPLE: Based on Beer-Lambert's law — absorbance is directly proportional to concentration within a specified range. Absorbance is measured at the λmax of the API. The calibration curve equation y = mx + c is used to calculate concentration. R² must be greater than 0.99. INSTRUMENTS: UV-Visible Spectrophotometer (Shimadzu UV-1800), Analytical Balance, Volumetric flasks (10ml, 100ml), Pipettes (1ml, 10ml), beaker, funnel, measuring cylinder. Solvent depends on the API — methanol is most common but may vary. STEP 1 — Average weight of tablets: Weigh 5 tablets. Average weight = Total weight divided by 5. STEP 2 — Weight equivalent calculation: Student selects target stock concentration (start with 1000 µg/ml). Weight equivalent = (Average weight × Target mg of API) divided by Label claim mg per tablet. The AI can calculate this if the student provides: average tablet weight, label claim, and target amount. STEP 3 — Prepare stock solution (1000 µg/ml): Accurately weigh calculated amount of tablet powder. Transfer to 25ml volumetric flask. Add approximately 20ml of solvent, dissolve and sonicate for 15 minutes. Make up volume to 25ml. This gives 1000 µg/ml stock solution. STEP 4 — Prepare intermediate solution (100 µg/ml): Pipette 1ml from 1000 µg/ml stock into 10ml volumetric flask. Make up to 10ml with distilled water. This gives 100 µg/ml. STEP 5 — Prepare working solutions (10 µg/ml), n=6: Pipette 1ml from 100 µg/ml solution into 10ml volumetric flask. Make up to 10ml with distilled water. Prepare 6 such solutions (S1-S6). This gives working concentration of 10 µg/ml. Note: Rarely, a direct 100 µg/ml working solution is used instead. STEP 6 — Measure absorbance: Measure absorbance of all 6 solutions at λmax of the API. Calculate average absorbance. CALCULATIONS (the AI can help if student provides values): Dilution factor = 25 × 10 × 10 = 2500 (for standard 3-step dilution). From calibration curve: x = (Average absorbance minus c) divided by m, where y = mx + c. Concentration in stock = x multiplied by dilution factor. Amount of API per tablet = (Concentration in stock × Average tablet weight) divided by Weight equivalent weighed. Percentage label claim = (Amount found divided by Label claim) × 100. ACCEPTANCE CRITERIA: The content of API in tablets should be not less than 95.0% and not more than 105.0% of the stated amount as per Indian Pharmacopoeia. Result must fall within 95-105% of label claim. INFERENCE: If percentage label claim is within 95-105%, the sample passes the assay test as per IP specifications and the UV spectrophotometric method is confirmed suitable for the assay of the tablet formulation.
