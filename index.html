Objective

Qualitative and quantitative analysis of substances by measuring absorbance in the UV and visible wavelength range.

Caution

1. Use clean scratch-free cuvettes and handle carefully. 2. Do not touch optical surfaces with bare hands. 3. Warm up the instrument before use for stable readings.

Responsibility

UG & PG Students

Operating Procedure

Step I: Switch ON power. Initialization takes approx. 3 min 30 sec.

Step II: Mode selection screen appears. Available modes: 1. Photometric, 2. Spectrum, 3. Quantitation, 4. Kinetic, 5. Time Scan, 6. Multicomponent, 7. Optional Program Pack, 8. Utilities.

Step III: Press Key 1–8 to select the required mode.

Step IV: The configuration screen for the selected mode is displayed.

Photometric Mode: Set wavelength (e.g. 550.0 nm). Press F1 to select % T or Abs. Press Go to WL Key to set wavelength. Press F3 to display measurement.

Spectrum Mode: Set Meas. Mode to Abs. Scanning range 190nm–1100nm. Rec. Range 3.999–3.999. Scan Speed: Fast. Number of Scans: 1. Display Mode: Sequential. Set baseline correction before scanning.

Quantitation Mode: Set up to 3 wavelengths (max 700nm). Method: C = K.Abs + B (K=1.000, B=0.0000). Unit: mg/ml. Number of measurements: 1. Enable baseline correction. Press Start Key to measure.

TROUBLESHOOTING — WRONG OR UNSTABLE READINGS (check in this order): 1. DILUTION/CONCENTRATION PROBLEM (most common): If absorbance is above 2 or going up to 4, the concentration is too high and outside Beer-Lambert law linear range. Ideal absorbance is 0.2 to 0.8. Must be below 1 for calibration and linearity studies. Solution: dilute the sample further and re-measure. This is the most frequent mistake students make. 2. CUVETTE ORIENTATION WRONG (very common): Cuvette has two sides — smooth (transparent, faces the light path) and rough (frosted, where you hold it). If inserted the wrong way the light cannot pass correctly and readings will be wrong. Always insert with smooth side facing the light beam. 3. WRONG CUVETTE TYPE: Lab has two types — plastic and quartz. Never use plastic cuvettes for UV measurements below 320nm. Plastic absorbs UV light and gives completely wrong readings and bad spectra. Always use quartz cuvettes for UV range measurements. 4. BASELINE NOT SET: Always zero the instrument with a blank (pure solvent with no drug) before measuring samples. If baseline is not set, all readings will be offset and incorrect. 5. FILTRATION PROBLEM: If performing an assay, ensure the drug solution is properly filtered before measurement. Unfiltered particles scatter light and cause incorrect or unstable absorbance values. This is rare but possible. 6. WARM-UP SKIPPED: Instrument requires 3 minutes 30 seconds warm-up after switching on for stable readings. Measuring before warm-up is complete causes unstable readings.
